Программа определения диапазона штрихкодов из массива

Программа предназначена для определения списка кодов из массива по конечным кодам диапазона.

Пример:

Мы имеем рулон напечатанных кодов с 1 по 1000, но не все коды из этого списка нанесены на продукцию а допустим с 10 по 450 и с 500 по 900. Чтобы не сканировать все нанесенные коды можно воспользоваться данной программой.
Для этого сканируем 10 код, затем 450, определяем коды входящие в диапазон, затем сканируем коды с 500 по 900 и также определяем список кодов входящих в диапазон.

Данная программа позволяет значительно сократить трудозатраты, но только в случае отсутствия выпадающих кодов внутри определяемых диапазонов.
